KOSMOS IHM

Interface Homme-Machine — Campagnes vidéo sous-marines

Application de bureau PyQt6 dédiée à la qualification, la validation et l'annotation de campagnes vidéo sous-marines stéréo et mono. Elle couvre l'ensemble du workflow opérationnel : ouverture de campagne, lecture vidéo, saisie d'événements, renseignement des métadonnées, visualisation GPS et export des livrables.

PyQt6 Python 3.11+ OpenCV Folium / Leaflet Paramiko SFTP Pandas

Page Accueil

Point d'entrée — identification du dérusher et ouverture de campagne

👤 Sélection du dérusher

  • Saisie du nom de l'opérateur avant toute ouverture
  • Nom mémorisé et affiché dans la toolbar pendant toute la session

📂 Ouverture de campagne

  • Sélection d'un dossier via l'explorateur système
  • Chargement automatique des vidéos MP4 présentes
  • Détection MONO / STEREO à l'ouverture
  • Déverrouillage de la navigation si des vidéos sont trouvées

⚙️ Connexion SFTP / Carte SD

  • Bouton "KOSMOS SD" dans la toolbar, disponible à tout moment
  • Connexion SSH avec IP, port, identifiants configurables
  • Choix libre du dossier distant (modifiable après connexion)
  • Listing récursif avec cases à cocher + sélection globale
  • Téléversement vers un dossier local avec barre de progression

Page Qualification

Revue des vidéos de campagne, carte GPS interactive et gestion de la corbeille

🌳 Arbre vidéo

  • Colonnes : nom, durée, FPS, résolution, taille, date
  • Filtre automatique des fichiers _stereo.mp4
  • Sélection → ouverture du player détaché et focus carte
  • Double-clic → plein écran immédiat

🗺️ Carte GPS Leaflet

  • Dialog QDialog flottant, indépendant de l'arbre
  • Marqueurs cliquables par vidéo avec popup enrichi
  • Popup : nom vidéo, coordonnées GPS, waypoint
  • Header de carte : nom campagne, ZONE, SITE
  • Tracé polyline GPS chronologique en bleu
  • Clic marqueur → sélection dans tous les arbres
  • Sélection arbre → pan/focus sur le marqueur
  • Rebuild automatique après édition des métadonnées

🗑️ Gestion de la corbeille

  • Arbre dédié aux vidéos supprimées (.trash/)
  • Restauration ou suppression définitive depuis l'IHM
  • Libération du verrou Windows avant suppression

👁️ Watcher système de fichiers

  • QFileSystemWatcher sur le dossier campagne
  • Mise à jour automatique si suppression/ajout externe
  • Debounce 600 ms pour éviter les rafraîchissements en cascade
  • Surveillance 2 niveaux max (pas les sous-dossiers .trash)

Page Métadonnées

Formulaires de saisie structurés, injection météo automatique et visualisation

📝 Formulaires dynamiques JSON

  • Champs générés dynamiquement depuis la structure JSON de la campagne
  • Types : texte libre, listes déroulantes, dates, coordonnées
  • Sauvegarde automatique avec debounce 800 ms après saisie
  • Sauvegarde forcée lors du changement de vidéo
  • Champs : survey_name, ZONE, SITE, GPS Waypoint, profondeur, etc.

🌤️ Injection météo automatique

  • Appel API météo au chargement de la campagne
  • Pré-remplissage des champs météo (vent, temp, état de mer…)
  • Basé sur les coordonnées GPS de la campagne

🖼️ Vignette vidéo

  • Extraction d'une frame à la position choisie
  • Affichage dans le panneau métadonnées
  • Traitement d'image (débrumage, égalisation histogramme)

🗺️ Sync carte en temps réel

  • Modification du GPS Waypoint → rebuild automatique de la carte
  • Sélection d'une vidéo → focus sur son marqueur
  • Ouverture automatique du player détaché à la sélection
  • Clic marqueur → sélection dans l'arbre

🌍 Métadonnées globales campagne

  • Champs partagés pour toutes les vidéos (survey_name, zone, site…)
  • Propagation vers le header de la carte Leaflet
  • Synchronisation entre pages

Fonctionnalités transversales

Disponibles sur toutes les pages — ergonomie, observabilité et connectivité

📊 Barre de statut campagne

  • Nombre total de vidéos chargées
  • Durée totale de la campagne (hh mm ss)
  • Compteur de vidéos validées (X/N · %)
  • Couleur adaptative : vert 100 % · jaune en cours · gris vide
  • Mise à jour en temps réel à chaque changement d'exploitabilité

🖥️ Plein écran vidéo

  • Double-clic sur le player pour passer en plein écran
  • Fenêtre dédiée avec affichage stéréo ou mono
  • Overlay "Appuyez sur Échap" affiché 3 s
  • Double-clic ou Échap pour quitter
  • Sortie automatique au changement de vidéo

🗺️ Synchronisation carte ↔ arbres

  • Clique dans n'importe quel arbre → focus sur le marqueur Leaflet
  • Clique sur un marqueur Leaflet → sélection dans tous les arbres
  • Fonctionne sur les 4 pages : Qualification, Validation, Événements, Extraction
  • Pan automatique de la carte sur la vidéo focalisée

⌨️ Raccourcis clavier

  • Espace — lecture / pause
  • ← / → — reculer / avancer de 5 s
  • Échap — quitter le plein écran
  • Actifs dès l'arrivée sur la page, sans clic préalable

🔭 Watcher de fichiers

  • Détection automatique des suppressions/ajouts externes
  • Tous les arbres mis à jour sans redémarrer l'application
  • Debounce 600 ms pour éviter les rafraîchissements multiples

📡 SFTP / Carte SD KOSMOS

  • Connexion SSH (Paramiko) avec IP/port/user/mot de passe
  • Dossier distant librement configurable (même après connexion)
  • Bouton "Actualiser" pour changer de dossier sans se reconnecter
  • Listing récursif avec cases à cocher hiérarchiques
  • Téléversement multi-fichiers avec progression et annulation

🔗 Callbacks cross-controller

  • on_video_focused(name) — arbre → carte (toutes pages)
  • on_metadata_saved() — rebuild carte si visible
  • on_video_selected(name, path) — métadonnées → player + carte
  • on_qualification_changed() — mise à jour barre de statut

🧵 Workers QThread

  • SftpConnectWorker — listing SFTP asynchrone
  • SftpDownloadWorker — transfert fichiers par fichier
  • WeatherWorker — appel API météo
  • ExportWorker — export vidéo OpenCV

⏱️ Debounce QTimer

  • 600 ms — watcher système de fichiers
  • 800 ms — sauvegarde métadonnées après frappe
  • 80 ms — focus JS sur la carte après show()
  • 0 ms (singleShot) — focus player après switch de page
